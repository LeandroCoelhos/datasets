Bank Marketing Data Set
Conjunto de dados de uma campanha de marketing de bancos brasileiros para predizer um depósito.
O conjunto de dados original encontra-se em:
http://archive.ics.uci.edu/ml/datasets/Bank+Marketing#

  [Moro et al., 2011] S. Moro, R. Laureano and P. Cortez. Using Data Mining for Bank Direct Marketing: An Application of the CRISP-DM Methodology. 
  In P. Novais et al. (Eds.), Proceedings of the European Simulation and Modelling Conference - ESM'2011, pp. 117-121, Guimarães, Portugal, October, 2011. EUROSIS.

  Available at: [pdf] http://hdl.handle.net/1822/14838
                [bib] http://www3.dsi.uminho.pt/pcortez/bib/2011-esm-1.txt
                
O intuito de disponibilizar aqui esse conjunto é manter um fácil e rápido acesso, sem passar por processos de compressão/descompressão do conjunto de dados exigidos pelo repositório.
